# CLI

pyredis-om ships with a single CLI command, `migrate`, that manages RediSearch
index creation, updates, and teardown from the terminal. No Python code
required — point it at your models module and it finds, hashes, and applies
schema changes.

## Quick start

```bash
pip install pyredis-om
export REDIS_OM_URL="redis://localhost:6379?decode_responses=True"
migrate --module myapp.models
```
